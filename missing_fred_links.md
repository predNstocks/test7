# Missing FRED Series Download Links

If a series failed to download automatically, use one of the links below to download the CSV manually.

- **T10Y2Y**: Graph CSV: https://fred.stlouisfed.org/graph/fredgraph.csv?id=T10Y2Y
  - Official API CSV (requires API key): https://api.stlouisfed.org/fred/series/observations?series_id=T10Y2Y&api_key=YOUR_KEY&file_type=csv
- **DGS10**: Graph CSV: https://fred.stlouisfed.org/graph/fredgraph.csv?id=DGS10
  - Official API CSV (requires API key): https://api.stlouisfed.org/fred/series/observations?series_id=DGS10&api_key=YOUR_KEY&file_type=csv
